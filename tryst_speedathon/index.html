Did you know?
The term "bug" was coined by Grace Hopper in 1947 when she discovered a moth trapped in a relay of the Harvard Mark II computer. This was the first documented computer bug.
Another one!

The HTTP 404 error, indicating that a web page cannot be found, is named after Room 404 at CERN, where the World Wide Web was born. This room housed the original web server.
